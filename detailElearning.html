Web Development Bootcamp

Deskripsi Materi

Dalam bootcamp ini, Anda akan belajar dasar-dasar hingga teknik lanjutan dalam pengembangan web, termasuk HTML, CSS, JavaScript, dan banyak lagi.

Materi yang Akan Dipelajari

  • Pengenalan HTML ▼
    • Tag dan Elemen HTML
    • Struktur Dasar HTML
    • Form dan Input HTML
  • Styling dengan CSS ▼
    • Selector dan Properti CSS
    • Layouting dengan Flexbox
    • Responsive Design
  • JavaScript Dasar ▼
    • Variabel dan Tipe Data
    • Fungsi dan Event Handling
    • Manipulasi DOM
  • Framework Front-end: React ▼
    • Pengenalan React
    • Component dan Props
    • State Management
  • Backend Development dengan Node.js ▼
    • Pengenalan Node.js
    • Membuat Server dengan Express
    • Database dengan MongoDB
